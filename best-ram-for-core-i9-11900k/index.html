How does RAM work?

RAM is intended for the short-term storage of data required by the PC for regular operation. However, unlike hard drives and SSDs, where data is stored indefinitely, RAM is cleared every time the system is rebooted. RAM only works when powered, distinguishing it from non-volatile storage devices such as hard drives and SSDs. Programs are loaded into RAM only for the duration of use, but at the same time, they are stored on drives permanently (until they are deleted).

What are the Important characteristics of RAM for i9-11900k?

Computers need quick access to temporary data to run programs and perform tasks. For example, modern computer games need to load graphic resources quickly. Games use RAM to read and write data because it is much faster than non-volatile storage devices.

Two factors included judging the characteristics of the RAM for i9-11900k.

  • Speed: Memory speed is measured in millions of transfer operations per second (MT / s), often confused with megahertz (MHz), although memory speed and clock speed are different. Higher memory speeds mean faster responsiveness to read and write requests and, therefore, better performance.
  • Capacity: It can be measured in gigabytes (GB). The chance to store more data applications when you have high storage capacity. With more RAM capacity, more applications can run simultaneously, and games can store more temporary data in it.

What kind of RAM is compatible with your motherboard?

Before you start thinking about RAM capacity and frequency, you need to make sure the RAM is compatible with your motherboard and your processor. Memory modules of the wrong type will not work, and if they do not meet the specifications, the memory performance will be lower than expected.

  • Module type

RAM is produced in the form of modules that are installed in connectors on the system board. RAM incompatible with your system will not fit or work properly. Modern motherboards support DDR4 memory modules. DDR4 memory should not be confused with DDR3 memory, which is the previous generation of SDRAM memory. These two types of memory modules are not interchangeable, and you cannot replace (for example) 8GB DDR3 memory with 16GB DDR4 memory.

  • DDR4 or SDRAM

Computers use random access memory (SDRAM). Synchronous DRAM is synchronized with the processor frequency. SDRAM has been continuously improved and now offers lower power consumption, faster data transfer rates, and more stable data transfers.

Modern computers use DDR4 or SDRAM memory as a standard. DDR4 – DDR technology is the fourth generation and has been replaced with SDRAM technology. DDR4 memory has increased data transfer rates, increased capacity, and lower operating voltage than previous-generation memory.

What speed of RAM do I need?

Find the right balance between capacity and speed. Chances are, 32GB of slow RAM won’t be ideal, as is 4GB of fast RAM. DDR4 memory speeds start at 1600MHz, but this is not fast by today’s standards. For example, the Intel Core i9-10900 processor supports 2933MHz memory in its base configuration.

In games, higher frequency RAM is beneficial. While the effect is not as noticeable as replacing the processor or graphics card, faster memory can also improve game performance and frame rates.

The performance improvement will vary from game to game; some games will run much faster, while others will be subtle. To assess the need for an update, you should look at the benchmark results with average frame rates.

In addition to raising the frame rate, faster memory can provide frame rate stability and faster frame loading. These figures will be presented in tests as low values ​​of 1% and 0.1% (average slowest indicator for 1% and 0.1% of recorded frames).

In addition to improving frame rates, fast RAM can improve game performance in other areas, including shorter loading times.
